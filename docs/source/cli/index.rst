duetector.cli
=========================================

``duectl``: CLI for start monitor, generate config.

``duectl-daemon``: Allow to run monitors and server as daemon.


``duectl-server``: CLI for start duetector server.

``duectl-server-daemon``: Allow to run server as daemon.
